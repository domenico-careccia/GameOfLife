# Conway's Game of Life (Fortran + MPI)

This small project is an implementation of the **[Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life)**.

The project is written in **Fortran 90**, with domain decomposition and parallelization using OpenMPI + CUDA. Although the Game of Life is a simplified discrete system, the parallelization techniques used here are foundational for more complex simulations.
